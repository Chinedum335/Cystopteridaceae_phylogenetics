# Cystopteridaceae_phylogenetics
This repository contains data, scripts, configuration files, and workflows for purc and homologizer analyses used in the article: Cystopteridaceae in focus: A model clade for studying polyploid systematics and macroevolution 

## Contents

- PURC processing workflows
- Iterative homologizer phasing analyses
- RevBayes scripts for MUL-tree inference
